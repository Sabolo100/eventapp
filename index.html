EventAI — Supabase Client

Welcome

This is a lightweight, static web app that talks directly to your Supabase database using only the Anon key.
Do not put the Service Role key, OpenAI key, or Google private key here.

Configure

Open config.example.js, copy to config.js, and fill SUPABASE_URL and SUPABASE_ANON_KEY. Optionally set BACKEND_URL for Make triggers.
